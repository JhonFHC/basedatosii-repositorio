Base de Datos II

SQL Avanzado, PL/SQL, Optimización y Administración

Este curso profundiza en el diseño, implementación y administración de bases de datos relacionales. Se enfoca en consultas complejas, procedimientos almacenados, triggers, optimización de queries y gestión de transacciones en entornos empresariales con Oracle, PostgreSQL y MySQL.

Explorar Unidades

Objetivos del Curso

• Consultas Avanzadas: Subconsultas correlacionadas, CTEs, Window Functions

• Programación en BD: PL/SQL, Stored Procedures, Funciones, Triggers

• Optimización: Índices, Planes de Ejecución, Tuning de SQL

• Administración: Usuarios, Roles, Backups, Recuperación

• Transacciones: ACID, Niveles de Aislamiento, Control de Concurrencia

Competencias Desarrolladas

• Diseño Lógico: Normalización hasta 5FN, Modelado dimensional

• Desarrollo Backend: Integración BD con aplicaciones (APIs)

• Tuning: Análisis de rendimiento y optimización de consultas

• Seguridad: Encriptación, SQL Injection Prevention, Auditoría

• Big Data: Introducción a bases de datos NoSQL (MongoDB)
